如何生成 PNG 图标

方法 1：在线生成器（推荐）

  1. 访问 RealFaviconGenerator
  2. 上传一张方形图片（建议 SVG 或高分辨率 PNG）
  3. 下载生成的图标包
  4. 提取 icon-192.png 和 icon-512.png
  5. 替换到 pwa-version/assets/icons/ 目录

方法 2：手动截取

  1. 使用浏览器的截图工具截取上方的 SVG
  2. 使用图片编辑工具（如 Preview, Photoshop）裁剪为 192x192 和 512x512
  3. 保存为 PNG 格式
  4. 替换到图标目录

图标准备好后，返回主页面继续开发
