TIB OS - 安装指南

将 TIB OS 添加到手机主屏幕，像原生 App 一样使用

iOS (iPhone/iPad)

1

在 Safari 浏览器中打开

确保使用 Safari 浏览器访问 TIB OS 网站

2

点击分享按钮

在屏幕底部工具栏，点击 ⎋ 分享按钮

3

选择"添加到主屏幕"

在弹出的菜单中，找到并点击"添加到主屏幕"选项

4

点击"添加"

确认应用名称和图标，然后点击右上角的"添加"按钮

5

完成！

现在可以在主屏幕看到 TIB OS 图标，点击即可像原生 App 一样使用

Android (Chrome)

1

在 Chrome 浏览器中打开

使用 Chrome 浏览器访问 TIB OS 网站

2

等待安装提示

Chrome 会自动显示"安装应用"提示（通常在地址栏右侧）

3

点击"安装"

确认应用名称和图标，然后点击"安装"按钮

4

完成！

现在可以在应用抽屉或主屏幕看到 TIB OS 图标

💡 小贴士

  • ✅ 确保使用 HTTPS（PWA 必须使用安全连接）
  • ✅ 安装后可以离线使用核心功能
  • ✅ 会像原生 App 一样全屏显示，没有浏览器地址栏
  • ✅ 可以接收推送通知（未来版本支持）

常见问题

Q: 没有看到安装提示？

A: Chrome 可能需要你先访问几次网站才会提示安装。你也可以点击 Chrome 菜单（⋮）→ "安装应用"

Q: iOS 上找不到"添加到主屏幕"？

A: 确保使用 Safari 浏览器，而不是 Chrome 或其他浏览器。Safari 底部工具栏中间有一个方形按钮（带箭头），那就是分享按钮。

Q: 安装后图标不显示？

A: 清除浏览器缓存，重新访问网站并重新安装。确保 manifest.json 中的图标路径正确。

Q: 可以从主屏幕删除吗？

A: iOS：长按图标 → 选择"移除书签"。Android：长按图标 → 卸载应用。
